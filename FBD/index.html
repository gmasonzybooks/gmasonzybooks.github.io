Ideas for accessibility.

Add a curve or vector, then double click on the curve or vector to open a dialog box for entering values directly.
The Curve dialog is not implemented.
The Vector dialog show how you could let users who were visually impared position vectors. The points locations were created in the Animator tool (see the XML). This aproach works for 2D and 3D images. The issue with this 3D approach is that there are multiple 3D directions that give the same 2D projection. However, there may be simple workarounds, or maybe it doesn't matter. A 3D example.
